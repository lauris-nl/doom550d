Doom 550D
=========

:Author: Bas Lichtjaar <doom@lauris.nl> and contributors
:License: GPL-2.0
:Summary: Experimental Doomgeneric port for the Canon EOS 550D.

The module expects an authorized Doom IWAD at::

    ML/DOOM/doom1.wad

Start it from the Magic Lantern Games menu.

Controls
--------

* Arrows: move and turn
* SET: fire
* Q: use/open
* INFO: Escape
* MENU: leave the game and restore the ML menu

This is an experimental engine build. Sound is disabled.
