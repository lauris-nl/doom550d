# Security Policy

Doom550D is experimental software that runs on a camera. Treat unexpected
behavior as potentially capable of disrupting camera operation or data access,
and use it at your own risk.

Report suspected security vulnerabilities privately to
[doom@lauris.nl](mailto:doom@lauris.nl). Include the affected version or commit,
impact, reproduction steps, and any relevant logs. Please do not open a public
issue for a vulnerability before it has been reviewed.

Report ordinary bugs and compatibility problems through GitHub Issues.
